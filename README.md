## 프로젝트 소개

부동산에 관심이 많아 부동산 관련 페이지를 제작해보고 싶어 뷰 문법으로 제작을 해보게 되었던 개인 뷰 프로젝트 입니다. 이슈가 생길때의 검색을 하다가 토스 결제 api를 만나게 되면서 결제 기능과 연동을 해보았습니다. 해당 프로젝트에 대해서는 뷰 문법의 이해와 응용력에 대하여 초점을 두게 되었으며, 백엔드까지는 제작을 안하게 되었지만, 최대한 실제 서비스를 할 수 있는 마음가짐으로도 제작하게 되었습니다. 해당 프로젝트는 상업적인 용도가 아니기 때문에 뉴스/이미지/아이콘 ETC 가져오게 되었습니다.

## 시작 가이드
### Requirements
For building and running the application you need:

- [Node.js 16.17.1](https://nodejs.org/ca/blog/release/v16.17.1/)
- [Npm 8.15.0](https://www.npmjs.com/package/npm/v/8.15.0)
- [Vue 3.2.13](https://www.npmjs.com/package/vue/v/3.2.13)
- [Sass13.2.1](https://www.npmjs.com/package/sass-loader)

### Installation
``` bash
$ git clone https://github.com/pdy1207/Vuebangapp.git 
$ cd vuebangapp
$ npm install
$ npm run serve
```
