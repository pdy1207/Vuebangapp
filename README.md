### Requirements
For building and running the application you need:

- [Node.js 16.17.1](https://nodejs.org/ca/blog/release/v16.17.1/)
- [Npm 8.15.0](https://www.npmjs.com/package/npm/v/8.15.0)
- [Vue 3.2.13](https://www.npmjs.com/package/vue/v/3.2.13)
- [Sass13.2.1](https://www.npmjs.com/package/sass-loader)

### Installation
``` bash
$ git clone https://github.com/pdy1207/Vuebangapp.git 
$ cd vuebangapp
$ npm install
$ npm run serve
```
